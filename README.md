# 🌍 Sarthi Travels - Travel Management System

**Sarthi Travels** is a full-stack travel management web application built using **PHP, MySQL, HTML, CSS, and JavaScript**. It is designed to help trip organizers efficiently manage trips, assign meals, track ingredients, and generate detailed reports.

## 🚀 Features

- 🧳 **Trip Management** – Create, edit, and delete trips with duration and payment tracking.
- 🍽️ **Meal Assignment System** – Assign breakfast, lunch, and dinner for each trip day.
- 🧂 **Ingredient Tracking** – Add ingredients with quantities and units per meal.
- 📄 **PDF Report Generation** – Export meal plans and ingredient lists to printable PDF.
- 🔐 **User Authentication** – Secure login, signup, and password recovery with validations.
- 🤖 **CAPTCHA Integration** – Protects login/signup forms from spam and bots.
- 🔍 **Search & Filter** – Easily manage and navigate through trip records.
- 🖥️ **Responsive UI** – Clean and mobile-friendly interface for smooth navigation.

## 🛠️ Technologies Used

- **Frontend:** HTML5, CSS3, JavaScript
- **Backend:** PHP (Core)
- **Database:** MySQL
- **Extras:** FPDF for PDF export, CAPTCHA for form security

## 🧑‍💻 Getting Started

1. Clone the repository:
https://github.com/aryanrajyaguru22/Sarthi_Travel.git

2. Import the SQL file to your MySQL server.

3. Configure your database connection in `includes/db.php`.

4. Run the project locally using WAMPP or any PHP-supported environment.
